Contact form

I have made some changes for the contact form validation. The validation used to go one by one on the errors. Now it displays all the errors at once. This sollution is better for the users, because they could see all their errors at once, instead of one. For the validation to pass it needs fill every critera for the form. When it passes it will send an alert, which tells the users that the message has been sent.

Correct navigation

The class "target" who tells the user which page they're on, was on the wrong element when you scrolled down to the game page. It looked like the users was still on the home page. I've added some javascript to fix this problem. To do that I added an eventlistener for when you scroll down to a certain point on the page. When you scroll down to that point the class "target" would be added to the element "games" instead. Also the "target" class on "home" will be removed.
